# Health Fact-Checker AI

A web-based AI tool built with **Gradio** and **Transformers** that validates common social media health claims as FACT or MYTH. 

## Features
- **Accurate Retrieval:** Uses a knowledge-base matching pattern to ensure factual accuracy.
- **Custom UI:** Professionally branded interface with a "Baby Blue" design.
- **Tech Stack:** Python, Gradio, Hugging Face Transformers.

## How to run
1. Clone this repository.
2. Install dependencies: `pip install -r requirements.txt`
3. Run the app: `python fact_checker.py`
